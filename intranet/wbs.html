The shared language

Cost codes & chart of accounts

Cost codes turn this WBS into numbers. Each division and line item maps to a cost code so estimates, purchase orders, invoices and the budget all roll up the same way — that's what lets us compare jobs and improve the next estimate.

BuilderK's codes follow the NAHB Standard Cost Codes & Chart of Accounts, the industry standard. The list above is the working structure; the official, fully-numbered NAHB code set is the reference we map to in app.builderk.com (Finance → cost codes / WBS).

🔗 Tied to the rest of the system
The same WBS feeds the Pre-Construction Checklist (finance setup), the budget in the app, and the schedule of values for lender draws.
