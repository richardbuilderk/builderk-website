Why this playbook exists

The Sudhagar Lesson

~$80,000
Left on the table at Sudhagar
6.7% of total project cost — recoverable on every future home

The Sudhagar project (14070 Kendale Lakes, Miami) was built on a 2,000 psf bearing assumption from a visual subgrade inspection — not a real geotechnical report. The actual limestone bearing was almost certainly 3,000–3,500 psf.

Result: 48" × 18" footings with #6 @ 6" o.c. — roughly 40% oversized. Combined with legacy CMU + LGS walls (not panelized) and no foundation-system evaluation, we left tens of thousands per home on the table.

This playbook fixes that. Three big levers: real geotech, matched foundation system, and panelized superstructure — applied per home type, per soil, per site.

Part 1 — Foundation Systems

The 9 Foundation Types

Each system, when it wins, and what it costs in dollars + days.

F1
▤ Mono-Slab

Mono-Slab + Thickened Edge

Single monolithic concrete pour combining slab + perimeter thickened-edge footing. Simplest fast-pour solution for inland sites with good bearing soil and no flood elevation requirement.

Cost
$6 – $10 / sqft
Install Time
5–7 days
Best Soil / Site
Inland uniform sand / limestone
Mechanistic Score
★★★★★
✔ Pros
  • ✓Single pour day — no separate footing cycle
  • ✓Lowest hard cost when soil + site allow
  • ✓Familiar to every FL concrete crew
⚠ Cons
  • ✗Requires good bearing soil (≥2,000 psf)
  • ✗No raised floor — limited for FEMA / drainage zones
  • ✗Still site-poured (lower mechanistic score)
Code: FBC residential, ACI 318
Vendors: Any FL concrete contractor
F2
⚒ Traditional CMU

CMU Stem Wall + Strip Footing

Florida's legacy default. Poured concrete strip footing + multi-course CMU stem wall + slab. High labor, multiple pour days, multiple inspections. Sudhagar was built this way.

Cost
$12 – $18 / sqft
Install Time
12–17 days
Best Soil / Site
Anywhere — but slow
Mechanistic Score
★★★★★
✔ Pros
  • ✓Familiar to every FL plan reviewer + inspector
  • ✓Works on any soil
  • ✓Lender / appraiser comfort
⚠ Cons
  • ✗12–17 day install (the slowest of all options)
  • ✗3–4 separate inspections required
  • ✗Scarce block mason labor
  • ✗No insulation (R-2 only)
  • ✗Termite-prone interface
Code: FBC residential, TMS 402/602
Vendors: Legacy — avoid going forward
F3
▦ ICF Stem

ICF Stem Wall + Mono-Slab

Insulated concrete forms (Fox Blocks, Nudura, BuildBlock) stacked for the stem wall, then poured monolithically with the slab. Hurricane-rated, insulated, and avoids the slow CMU labor problem.

Cost
$10 – $14 / sqft
Install Time
7–10 days
Best Soil / Site
Inland good bearing + hurricane priority
Mechanistic Score
★★★★★
✔ Pros
  • ✓200+ mph hurricane wind rating
  • ✓R-22 insulation built in (vs R-2 for CMU)
  • ✓Faster than CMU (7–10 vs 12–17 days)
  • ✓Mexico Beach 'Miracle House' survived Hurricane Michael Cat-5
  • ✓Growing FL adoption with established trades
⚠ Cons
  • ✗Higher material cost than CMU upfront
  • ✗Requires ICF-trained installer (smaller pool)
  • ✗Single concrete pour day is critical — no second chance
Code: FBC residential, ICC-ES
Vendors: ICF ICF LLC (Tampa/Orlando), Coastal ICF (Tampa Bay), Florida ICF Systems (Palm City), McDaniel Construction (Ft Myers)
F4
▥ Precast Stem

Precast Stem Wall (Superior Walls type)

Factory-cast concrete wall panels with integral footer beam, dropped onto a compacted crushed-stone bed. No site-poured footing. Walls + structural footer arrive on a truck and are crane-set in hours.

Cost
$11 – $16 / sqft
Install Time
1–2 days
Best Soil / Site
Inland good bearing
Mechanistic Score
★★★★★
✔ Pros
  • ✓Entire foundation walls installed in 1 day
  • ✓5,000 PSI fiber-reinforced concrete (vs 2,500–3,000 PSI field pour)
  • ✓Insulation + waterproofing integrated
  • ✓No concrete cure window — frame the next day
⚠ Cons
  • ✗Limited FL adoption — most dealers are northeast-based
  • ✗Designed for basements; FL stem-wall use is a partial fit
  • ✗Freight cost from out-of-state plants
  • ✗FL plan reviewers less familiar — adds permit review time
Code: ICC-ES, IRC, FBC residential
Vendors: Superior Walls / Weaver Precast (verify FL dealer status), Coreslab Structures (FL)
F5
⚓ Precast Piers

Precast Concrete Pier Foundation

Discrete cylindrical concrete piers with steel anchor plates on top, set in a compacted gravel pad. The home (modular box or panelized walls) bolts directly to the pier anchors. Industry standard for modular / manufactured / elevated structures.

Cost
$4 – $8 / sqft
Install Time
1–2 days
Best Soil / Site
Modular / ADU / elevated only
Mechanistic Score
★★★★★
✔ Pros
  • ✓Cheapest 'real' foundation ($4–8 / sqft)
  • ✓Installs in 1–2 days
  • ✓Lifts home above FEMA Base Flood Elevation cleanly
  • ✓Allows airflow under home (hurricane debris washes through)
  • ✓Standard for the modular / HUD-code industry
⚠ Cons
  • ✗Not suitable for site-built custom 2-story homes (lender / appraiser pushback)
  • ✗Open underside = more termite + moisture attention
  • ✗Perceived as 'manufactured-grade' in high-end market
Code: FBC residential, HUD code (manufactured)
Vendors: Local precast or sonotube drilled-pier contractors statewide
F6
⌁ Helical Piles

Helical Pile + Precast Grade Beam

Steel piles screwed (not driven) into the ground by a hydraulic torque head. Bearing plates on top connect a precast or cast-in-place grade beam. Torque-controlled, instrumented install gives real-time capacity reading per pile.

Cost
$8 – $14 / sqft
Install Time
1 day
Best Soil / Site
Universal — variable, fill, sloped, lake-adjacent, karst
Mechanistic Score
★★★★★
✔ Pros
  • ✓🏆 The most universal mechanistic foundation
  • ✓1 day install for typical home (16–24 piles)
  • ✓Zero concrete cure for load-bearing element
  • ✓Works in variable soils, fill, sloped lots, lake-adjacent, mild karst, organic muck
  • ✓Each pile gets a verified capacity reading (torque-to-load correlation)
  • ✓Pairs perfectly with panelized superstructure
  • ✓$5,000–$12,000 total pile cost for a typical home
⚠ Cons
  • ✗Coastal V-zone requires prestressed concrete piles instead
  • ✗Heavy karst with large voids may need steel pipe to bedrock
  • ✗Requires experienced installer + engineer familiar with helical design
Code: ICC-ES ESR-3032, FBC, ASCE 7-22
Vendors: Magnum Piering, Ram Jack Florida, ATLAS Resistance Piers
F7
≈ Prestressed Piles

Prestressed Concrete Pile + Precast Cap

Factory-cast prestressed concrete piles driven on site by an impact hammer, capped with precast or cast-in-place pile caps. Mandatory for coastal V-zones and the standard for FL coastal high-hazard construction.

Cost
$20 – $35 / sqft
Install Time
5–7 days
Best Soil / Site
Coastal V-zone, marine, deep poor soils
Mechanistic Score
★★★★★
✔ Pros
  • ✓Code-required for V-zones and coastal high-hazard
  • ✓Handles marine bearing and deep poor soils
  • ✓Highest capacity per pile (50–100+ tons)
  • ✓FL vendors well-established with HVHZ approval
⚠ Cons
  • ✗Most expensive foundation option ($20–$35/sqft)
  • ✗Requires pile-driving rig (noise, vibration, access constraints)
  • ✗5–7 day install (driving + capping)
  • ✗Heavy logistics (cranes, trucks)
Code: FBC, ACI 318, ASCE 7-22, Miami-Dade NOA for HVHZ
Vendors: Paradise Precast (SW FL), DLS Prestressed (S FL), Florida Concrete Products (Miami)
F8
⛏ Auger-Cast

Auger-Cast Piles + Grade Beam

Continuous flight auger drills into the ground; concrete pumped under pressure through the hollow stem as the auger is withdrawn. Steel cage inserted before concrete sets. Quieter and lower-vibration than driven piles.

Cost
$15 – $25 / sqft
Install Time
5–7 days
Best Soil / Site
Restricted access, sensitive sites, high water table
Mechanistic Score
★★★★★
✔ Pros
  • ✓Low noise + low vibration (sensitive neighborhoods)
  • ✓Works in restricted-access sites
  • ✓Effective in high water table conditions
  • ✓No spoil concerns — soil is displaced laterally
⚠ Cons
  • ✗Specialized equipment + crew (small FL supply)
  • ✗Less mechanistic — still site-poured concrete
  • ✗Requires real-time monitoring during install
  • ✗Quality control tied to operator skill
Code: FBC, ACI 318
Vendors: Specialty geotechnical contractors — case-by-case
F9
⛰ Steel Piles

Steel Pipe Pile + Cap

Steel pipe or H-pile sections driven to refusal in bedrock. Cap is bolted to pile top, eliminating the cure window for the structural element. Ideal for karst-prone South Florida limestone with cavities or voids.

Cost
$18 – $30 / sqft
Install Time
3–5 days
Best Soil / Site
Karst with voids, deep limestone
Mechanistic Score
★★★★★
✔ Pros
  • ✓Goes through voids/cavities to reach competent bedrock
  • ✓Bolt-on cap = no concrete cure for structural piece
  • ✓Highest 'mechanistic' rating among pile options
  • ✓Predictable capacity (driven to refusal)
⚠ Cons
  • ✗Steel cost volatility
  • ✗Requires impact hammer / hydraulic driver
  • ✗Corrosion protection needed in some soils
  • ✗Higher cost than helical or concrete piles
Code: FBC, ASCE 7-22
Vendors: Specialty pile contractors — coordinate with EOR

Part 3 — Multifamily Playbook

Apartment Complexes — Different Scale, Different Math

Multifamily / apartment construction has its own foundation playbook — heavier loads, denser column grids, taller structures, and scale economics that turn small per-unit savings into seven-figure project deltas.

3 Additional Foundation Types for Multifamily

These complement the 9 home foundations above and are specific to MF building loads.

F10Multifamily

Mat Slab / Raft Foundation

Single thick reinforced concrete slab spanning the entire footprint. Distributes load across weak or variable soils. Standard for 4–8 story podium MF and structures on poor bearing.

Best for
4–8 story MF on weak/variable soils, podium buildings, where pile counts would be uneconomical
Cost
$15 – $25 / sqft of footprint
Time
10–15 days (single huge pour day)
Pros
  • ✓Distributes load uniformly
  • ✓Eliminates differential settlement
  • ✓Works on bad soils
  • ✓Code-familiar to MF reviewers
Cons
  • ✗High concrete volume = high cost
  • ✗Long pour day (massive concrete order)
  • ✗Curing time before vertical work
  • ✗Site-poured (low mechanistic score)
F11Multifamily

Pile Group + Pile Cap Grid

Clusters of 2–6 piles per column location, capped by precast or cast-in-place pile caps, tied together by grade beams. The standard for 5-over-1 podium MF in soft soils or coastal zones.

Best for
5-over-1 podium MF, coastal MF, MF on organic/muck, taller buildings (6+ stories)
Cost
$22 – $40 / sqft of footprint
Time
10–18 days (drive piles + cap + grade beams)
Pros
  • ✓Handles heaviest MF loads
  • ✓Mandatory for coastal V-zone MF
  • ✓Predictable per-pile capacity
  • ✓Standard MF design pattern
Cons
  • ✗Most expensive foundation option
  • ✗Pile driving = noise + vibration
  • ✗Crane-heavy logistics
  • ✗Permits + neighbor coordination
F12Multifamily

Drilled Caissons / Shafts

Large-diameter (24"–60") drilled shafts with reinforced concrete, bearing on competent stratum or rock. Used for taller MF (7+ stories) and where pile driving isn't feasible (urban infill, sensitive sites).

Best for
Mid-rise to high-rise MF (7–15 stories), urban infill, sensitive surroundings, when bedrock is deep
Cost
$25 – $50 / sqft of footprint
Time
12–20 days
Pros
  • ✓Highest capacity per element (100+ tons)
  • ✓Quiet vs. driven piles
  • ✓Works for urban infill MF
  • ✓Reaches deep bearing strata
Cons
  • ✗Specialty equipment + crew
  • ✗High cost
  • ✗Slow individual install
  • ✗Real-time inspection required

Decision Flow

How to Pick — Step by Step

⛏
Step 1

Order the geotech FIRST

No foundation design starts without a sealed geotechnical report for any project hitting trigger criteria (see Geotech Policy below). Cost: $2K–$3.5K. ROI: 4–6×.

⚲
Step 2

Identify the SITE condition

Coastal V-zone? → Stack B. Lake/wetland adjacent or variable soils? → Stack A. Karst? → F9. ADU/cottage? → Stack C.

⌂
Step 3

Identify the PRODUCT type

Custom 2-story home? Stack A or B. ADU or modular? Stack C. Multifamily? Helical field + panelized + pods.

🛡
Step 4

Confirm with the EOR + Andrés peer review

Footing schedule reviewed before permit submission. No exceptions on major homes.

⚡
Step 5

Order foundation + panels in parallel

Foundation arrives day 1. Panels arrive day 2. Crane set both. Dried-in in days, not weeks.

⚠ Geotech Policy — Mandatory Before Foundation Design

Effective immediately for any project hitting any one trigger.

When a full geotech is required (any one trigger):

  • ✓Two-story or taller
  • ✓A/C area > 2,500 SF
  • ✓Within 200 ft of lake / canal / wetland
  • ✓HVHZ (Miami-Dade / Broward)
  • ✓Coastal V / A / Coastal A zone
  • ✓Variable soils (fill, karst, organic)
  • ✓Pile / deep foundation considered
  • ✓Project hard cost > $400K

Minimum geotech scope:

  • ◎2–4 SPT borings to 30–40 ft
  • ◎Blow counts every 5 ft
  • ◎Lab tests (Atterberg, moisture, gradation)
  • ◎Seasonal high water table measured
  • ◎Bearing capacity + safety factor
  • ◎Settlement analysis
  • ◎Shallow-vs-pile recommendation
  • ◎Sealed by FL-licensed Geotech PE

Approved vendors:

Task Laboratories (Miami, 305-235-5355) · Universal Engineering Sciences (statewide) · Terracon (statewide) · Nutting Engineers (Miami-Dade) · GFA International (multi-FL)

Cost: $2,000–$3,500 per site  ·  ROI: 4–6× minimum  ·  Saves: $8K–$15K in over-designed concrete + rebar per home

Soil & Site Conditions Across FL

▣

South FL (Miami-Dade, Broward)

Soil: Limestone bearing + karst risk

Recommended: F6 Helical OR F9 Steel pile (if karst severe)

≈

Coastal V-Zones (Keys, FMB, Panhandle beaches)

Soil: Sandy + marine bearing

Recommended: F7 Prestressed pile + cap (mandatory)

⛰

Central FL (Orlando, Lakeland, Lake County)

Soil: Inland sandy + some limestone

Recommended: F6 Helical or F3 ICF stem

≋

Everglades fringe / Wetlands (W Miami-Dade, Collier)

Soil: Organic muck + high water

Recommended: F6 Helical to refusal

≈

Lake/Canal-adjacent (statewide)

Soil: High water table + variable

Recommended: F6 Helical + raised structure

⛰

Panhandle / NW Florida

Soil: Sandy + some clay + sloped lots

Recommended: F6 Helical with stepped grade beam

≈

Treasure Coast / Atlantic

Soil: Sandy coastal + A-zone

Recommended: F7 Pile or F3 ICF stem (depending on zone)

⌂

Workforce / Production lots

Soil: Varies — usually inland sandy

Recommended: F5 Pier + S6 Modular
