The plan

Schedule & Last Planner

We plan with the Last Planner System — collaborative, commitment-based planning that the people doing the work help build. It layers on top of the master schedule.

  • Master schedule (CPM) — the full project critical path in BuilderK.
  • Reverse pull plan — built backward from the milestone with the trades, so handoffs are real.
  • 6-week lookahead — make-ready planning; clear constraints (design, materials, prereq work, permits) before a task is due.
  • Weekly Work Plan (WWP) — only ready work is committed; we measure what we promised vs. delivered.
  • PPC (Percent Plan Complete) — commitments met ÷ commitments made. Target ≥ 80%; every miss gets a reason so we fix the system.
